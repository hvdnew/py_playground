
# Tree
- One and Only one Root
- Each node will have at most 1 parent
- Multiple children

# Binary Search Tree
- Left smaller than root
- Larger than root

## Types of tree traversal
- Pre Order 
    - Root, Left, Right
- In Order
    - Left, Root, Right | Start with the smallest (BST) and go to the largest
- Post Order
    - Left, Right, Root |
